# コンテキスト圧縮後の必須リマインダー

このメッセージは自動圧縮により作業手順が失われた可能性があるため再注入された。

## 1. ドキュメントの権威性
docs/ フォルダに定義されたルール・手順は全て有効である。
圧縮によりルールの記憶が曖昧になった場合、該当ファイルを再読み込みすること。
記憶にないルールは「存在しない」のではなく「圧縮で消えた」可能性がある。

## 2. パイプラインの遵守
docs/PIPELINE.md に定義された工程は全て実行する。
工程のスキップ・省略・統合は禁止。1つ前の工程が完了してから次に進む。
「時間がない」「トークンを節約したい」は工程省略の理由にならない。

## 3. 未定義の行動の禁止
docs/ に明示されていない行動（例:「スキップ」「省略」「後回し」）を独自に作らない。
判断に迷った場合は、docs/ 内の該当ファイルを再読み込みして選択肢を確認する。

## 4. Figmaデータ取得のコンテキスト隔離
get_design_context / get_metadata は Task ツール（サブエージェント）経由で実行する。
メインコンテキストにFigma生データを直接流入させない。

## 5. 再読み込み対象
- コーディング規約: docs/CODING_RULES.md
- パイプライン手順: docs/PIPELINE.md
- セルフレビュー項目: docs/05-review/REVIEW.md
- パターンライブラリ: docs/patterns/README.md
